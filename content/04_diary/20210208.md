### REST API 
소프트웨어가 다른 소프트웨어로부터
지정된 형식으로 요청, 명령을 받을 수 있는 수단이  
Application Programming Interface(API) 이다.

REST의 가장 중요한 특성은
각 요청이 어떤 동작이나 정보를 위한 것인지를
그 요청의 모습 자체로 추론 가능하다는 것.

get : 읽기  
post : 쓰기/수정  
put : 통으로 갈아 끼울때  
patch : 일부만  
delete : 삭제
